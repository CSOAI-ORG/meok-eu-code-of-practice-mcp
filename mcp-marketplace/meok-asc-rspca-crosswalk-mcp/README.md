## Why this exists

Sainsbury's is at 100% ASC for farmed seafood. Co-op pledged the same by 2027. Tesco Finest and M&S Select Farms still gate on RSPCA Assured. GlobalG.A.P. IFA Aquaculture v6 is the only farm-level standard with GFSI + GSSI recognition for export.

If you supply more than one retail channel, you do not have one audit. You have three, with overlapping evidence, asked in different shapes. This MCP collapses that to one pack — and prints the topic-by-topic mapping that proves you have it.
