## Cost Reduction Strategies

### 1. Hybrid Sourcing Approach (RECOMMENDED)
- **Buy from AliExpress:** Servos, sensors, basic electronics (~£70)
- **Buy from UK:** Raspberry Pi, filament, urgent items (~£35)
- **Total Estimated:** £105

### 2. Bulk Buying (If Building Multiple)
- AliExpress 10+ servos: 15-20% discount
- Filament 3+ rolls: 10-15% discount

### 3. Alternative Parts

| Premium Part | Budget Alternative | Savings |
|--------------|-------------------|---------|
| DS3218 Servos (£4.50) | MG996R (£2.50) | £2 per servo |
| Arduino Uno (£12) | Arduino Nano (£3) | £9 |
| Raspberry Pi 4 (£55) | Raspberry Pi Zero (£15) | £40 |
| Metal Gear Servos | Standard Plastic Gear | 50% less |
| Pre-made Frame | Self-printed PETG | £30-50 |

### 4. Free/Cheap Alternatives

- **Frame:** Print yourself vs buying pre-made (saves £30-50)
- **Power:** USB power bank initially vs dedicated battery (saves £10)
- **Camera:** Use old phone camera with IP camera app (free)
- **Sensors:** Start with basic IMU, add more later
