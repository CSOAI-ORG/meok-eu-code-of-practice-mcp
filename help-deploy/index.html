FAQs

How do I get a free API key?

Sign up at meok-attestation-api.vercel.app/signup. The key arrives in your email instantly.

What's the difference between Free and Pro?

Free: 200 calls/day per MCP, no signed Ed25519 receipts. Pro: 1,000 calls/day per MCP, signed Ed25519, public verify URLs, priority support. £199/mo.

How does the offline verify tool work?

pip install meok-attestation-verify then meok-attestation-verify < cert.json. Returns {valid: true/false, message: ...}.

Can I get a refund?

Subscriptions: pro-rated refund within 14 days. One-time purchases: refundable within 7 days if not yet delivered.
