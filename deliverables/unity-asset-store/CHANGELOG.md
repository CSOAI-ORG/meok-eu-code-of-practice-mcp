# Changelog

## [0.2.0-phase2] - 2026-05-30
### Added
- Full CSOAIClient with MCP, A2A, RegGeoInt, Trust Layer, and Marketplace APIs
- Compliance map integration for deployment planning
- Trust score querying for AI characters
- Character marketplace browse and purchase
- Server card auto-discovery
- Unity 2021.3 LTS+ support
- WebGL compatibility

### Features
- `HealthAsync()` — Server health check
- `ListToolsAsync()` — MCP tool discovery
- `CallToolAsync()` — Execute governance tools
- `GetAgentCardAsync()` — A2A agent discovery
- `GetComplianceMapAsync()` — 50+ jurisdiction data
- `CrossBorderAdvisoryAsync()` — Deployment compliance checks
- `GetTrustScoreAsync()` — Character trust scoring
- `ListMarketplaceAsync()` — Browse AI characters
- `PurchaseCharacterAsync()` — Marketplace transactions

## [0.1.0] - 2026-05-15
### Added
- Initial CSOAI SDK scaffold
- Basic HTTP client for MEOK Protocol Nexus
- MCP tool call support
- A2A task sending
