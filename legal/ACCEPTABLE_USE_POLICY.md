# Acceptable Use Policy — MEOK AI Labs

**Last Updated:** 2026-05-09

## 1. Purpose

This Acceptable Use Policy ("AUP") sets out the rules for using MEOK AI Labs' Services, including websites, MCP servers, APIs, and compliance tools. By using our Services, you agree to comply with this AUP.

## 2. Prohibited Uses

You may not use the Services for:

### 2.1 Illegal Activities
- Any activity that violates applicable laws or regulations
- Fraud, money laundering, or financial crimes
- Intellectual property infringement
- Export control violations

### 2.2 Harmful Content
- Harassment, bullying, or threats
- Hate speech or discrimination
- Doxxing or sharing private information without consent
- Content that exploits or harms minors

### 2.3 Security Violations
- Unauthorised access to systems or data
- Distribution of malware, viruses, or harmful code
- Denial of service attacks
- Credential stuffing or brute force attacks
- Circumventing security or authentication measures

### 2.4 Service Abuse
- Excessive API calls beyond rate limits
- Scraping or data mining outside provided APIs
- Creating multiple accounts to abuse free tiers
- Reselling Services without authorisation

### 2.5 Misrepresentation
- Impersonating others or misrepresenting your affiliation
- Submitting false compliance attestations
- Forging HMAC signatures or verification data

## 3. MCP Server Usage

When using our MCP servers:

### 3.1 Rate Limits
Respect published rate limits. Excessive usage may result in throttling or suspension.

### 3.2 API Keys
- Do not share API keys with unauthorised parties
- Do not embed API keys in client-side code
- Rotate compromised keys immediately

### 3.3 Tool Usage
- Use tools only for their intended purpose
- Do not use compliance assessment tools to generate false or misleading reports
- Do not automate the submission of false data

### 3.4 Freemium Tiers
Freemium access is provided for evaluation and low-volume use. Systematic abuse of free tiers (e.g., creating multiple accounts) will result in termination.

## 4. Compliance Content

### 4.1 No Legal Advice Disclaimer
Our Services provide technical compliance assessments. Outputs are not legal advice. You must not represent MCP-generated content as legal opinion.

### 4.2 Attestation Integrity
- You must not falsify or tamper with HMAC-signed attestations
- Verification through meok-attestation-api.vercel.app is the authoritative source
- False attestations may be reported to relevant authorities

## 5. Enforcement

### 5.1 Monitoring
We reserve the right to monitor usage of the Services to ensure compliance with this AUP.

### 5.2 Consequences of Violation
Violations may result in:
- Warning
- Temporary suspension of access
- Permanent termination of account
- Revocation of API keys
- Reporting to law enforcement where appropriate
- No refund for terminated accounts

### 5.3 Reporting Violations
Report violations of this AUP to: abuse@meok.ai (will be configured upon domain setup)

## 6. Changes

We may update this AUP at any time. Continued use of the Services after changes constitutes acceptance.
