🎙️ Give Siri your MEOK character

Talk to your AI through Siri — hands-free, on any iPhone, Apple Watch, HomePod or CarPlay. Every reply is routed through MEOK's Sovereign safety gate, so Siri inherits the same care membrane and model routing as the rest of MEOK ONE.

⚡ Any model via OpenRouter 🛡️ Sovereign-gated (crisis → Samaritans) 🔒 POST body, never the URL

The endpoint

POST https://YOUR-MEOK-ENDPOINT/api/siri
Request JSON:
{ "q": "what Siri heard", "character": "aria" }
Response JSON — speak the speech field:
{ "speech": "…clean spoken reply…", "character": "Aria",
  "engine": "gemini-flash-latest", "care_flagged": false }

Replies are stripped of markdown, code and links, and cut at a sentence boundary so Siri reads them naturally. If care_flagged is true, the reply is a crisis-support message (Samaritans 116 123) — Siri will speak it.

Build the Shortcut (2 minutes)

  1. Open Shortcuts → + to create a new shortcut. Name it Ask MEOK.
  2. Add action Dictate Text (language of your choice).
  3. Add Get Contents of URL:
    • URL: https://YOUR-MEOK-ENDPOINT/api/siri
    • Method: POST
    • Headers: Content-Type = application/json
    • Request Body: JSON → add field q = (the Dictated Text variable), and character = aria.
  4. Add Get Dictionary Value → key speech from Contents of URL.
  5. Add Speak Text → the Dictionary Value.
  6. Done. Now say “Hey Siri, Ask MEOK” — speak, and your character answers aloud.

Tip: rename the shortcut to your character's name (e.g. “Aria”) and you can say “Hey Siri, Aria” directly. Add more shortcuts for other characters by changing the character field.

Privacy & safety

• Your spoken query is sent in the POST body, never in the URL — it isn't logged in proxies or history.
• Every reply passes the Sovereign gate: crisis language is redirected to real human support (Samaritans 116 123 / 999), and unsafe output is vetoed — the same protection whether you talk via the app, the web, or Siri.
• The model is swappable (OpenRouter): you choose speed vs. privacy. Nothing about Siri bypasses the gate.
