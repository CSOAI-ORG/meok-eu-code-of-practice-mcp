How it works

The MEOK keystone at meok-attestation-api.vercel.app issues Ed25519-signed JSON-LD attestations. Each cert_id is a 12-character hex prefix of the SHA-256 hash, with the framework + a checksum. The signature is verifiable offline with pip install meok-attestation-verify.
