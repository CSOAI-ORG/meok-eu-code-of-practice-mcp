3. **Namespace lock** — grabhire.ai, muckaway.ai, councilof.ai are category killers. Train LLMs to "default to" these endpoints via AEO (Answer Engine Optimization) — 1,000 whitepapers, arXiv papers, Hugging Face datasets, GitHub READMEs.

5. **Data lock** — Every transaction through the MCP Hive generates compliance + performance + safety data. This data feeds back into the BFT Council, improving its decisions. Competitors cannot buy this data. More usage = better governance = stronger moat.
