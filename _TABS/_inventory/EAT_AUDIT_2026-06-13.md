## Apify — the critical miss, expanded

**Why this matters:** Apify has a real, browseable MCP server store at `apify.com/store/categories/mcp-servers`, and 5,000+ Actors indexed. AI agents pull from it via the Apify MCP server (`mcp.apify.com`) — the same channel is integrated into Claude Desktop, Cursor, VS Code, ChatGPT. Zero MEOK coverage = we miss the only channel that:
1. Has built-in billing (agentic payments via x402 / Skyfire)
2. Is a default MCP server in every major AI client
3. Generates recurring revenue per Agent invocation

**Path to Apify listing per package:**

```bash
# Per package — needs an Actor wrapper that calls our existing server.py
# Container pattern (matches our existing Dockerfile.hardened):
mkdir .actor
cat > .actor/actor.json <<EOF
{
  "actorSpecification": 1,
  "name": "meok-eu-ai-act-compliance",
  "title": "MEOK EU AI Act Compliance",
  "description": "410 articles from EUR-Lex via FTS5 search. Instant risk scan, deterministic class detection, evidence pack export. By MEOK AI Labs.",
  "version": "1.0.0",
  "input": "./input_schema.json",
  "dockerfile": "../Dockerfile.hardened"
}
EOF

# Apify push:
apify login
apify push
```

**Time per Actor:** ~1 hour to wrap server.py + test locally + push to Apify.
**Total for 14 top compliance packages:** ~14 hours agent + $5-50/mo per Actor run costs.
