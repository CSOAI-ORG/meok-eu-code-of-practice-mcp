## FAQ

**Q: What is an MCP server?**
A: Model Context Protocol (MCP) is a standard that lets AI assistants like Claude connect to external tools and data. An MCP server exposes your business logic as tools that AI can use.

**Q: What AI assistants work with MCP?**
A: Claude Desktop, Cursor, VS Code with Claude extension, and any MCP-compatible client.

**Q: How long does delivery take?**
A: Basic: 3-5 days. Standard: 5-7 days. Premium: 7-14 days.

**Q: Can you integrate with my existing API?**
A: Yes. REST APIs, GraphQL, databases (PostgreSQL, MongoDB, SQLite), file systems, and custom protocols.
