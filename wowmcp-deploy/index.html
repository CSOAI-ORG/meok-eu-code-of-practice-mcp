The 10 tools

raid_strategy_lookup

Pull current raid strat, boss mechanics, cooldown timings for any tier.

character_analyse

Parse armory data — gear, talents, parses, recent logs.

auction_house_query

Real-time AH prices, trends, market depth for any commodity.

mythic_plus_route

Optimised M+ routes, affix strategies, timer targets.

gear_compare

achievement_check

Latest patch notes, hotfixes, datamined upcoming changes.
