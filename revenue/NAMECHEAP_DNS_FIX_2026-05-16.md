# Namecheap DNS Fix — 2026-05-16

Two domains are DEAD per the 2026-05-16 audit:
1. **optomobile.ai** — DNS resolution fail
2. **agriculture-robotics.ai** — DNS resolution fail

## Root cause (most likely)

Nameservers point at expired Manus hosting OR were set to Namecheap default and never updated when other domains migrated to Vercel.

## Recovery — 10 minutes, Nick clicks

### Per-domain steps

1. Sign in: **namecheap.com → Account → Domain List**
2. Find **optomobile.ai** → click **MANAGE**
3. Scroll to **NAMESERVERS** section
4. Dropdown: select **Custom DNS**
5. Set entries to:
   ```
   ns1.vercel-dns.com
   ns2.vercel-dns.com
   ```
6. Click the green ✓ to save
7. Repeat all steps for **agriculture-robotics.ai**

### Then add domains to Vercel project

```bash
cd ~/clawd/meok/ui
vercel domains add optomobile.ai --project meok
vercel domains add agriculture-robotics.ai --project meok
```

Vercel will prompt to update DNS — since we already set nameservers to Vercel, this is automatic. It will show "Valid Configuration" within 5-30 mins.

### Verify (do after 30 minutes)

```bash
dig +short optomobile.ai
dig +short agriculture-robotics.ai
# Expect Vercel IPs (76.76.x.x range)

curl -sI https://optomobile.ai/ | head -3
curl -sI https://agriculture-robotics.ai/ | head -3
# Expect HTTP/2 200 or 307 redirect
```

## If DNS still fails after 30 minutes

### Check 1: domain expiry
```bash
whois optomobile.ai | grep -iE "expir|status"
whois agriculture-robotics.ai | grep -iE "expir|status"
```
If `Expiration Date` is past today → renew at Namecheap (£35–60/yr per .ai)

### Check 2: nameservers set correctly
```bash
dig +short NS optomobile.ai
dig +short NS agriculture-robotics.ai
# Should show ns1.vercel-dns.com / ns2.vercel-dns.com after step 5
# If still showing old NS, Namecheap save didn't apply — repeat steps 4-6
```

### Check 3: Vercel verification
- Vercel dashboard → meok project → Settings → Domains
- Each domain should show "Valid Configuration" green badge
- If "Invalid Configuration" → click `Refresh` button, wait 5 min
