## How to list (Sedo specifically)

1. Login to https://sedo.com (or sign up — free)
2. "Sell domains" → "Add a domain"
3. Enter the domain
4. Pick "Verify by HTML upload" → grab the verification file → upload to your domain root (Vercel-hosted ones already have a `public/` dir; Cloudflare ones need a different path)
5. Set BIN price + minimum offer
6. Paste the description below
7. Pick keywords from the keyword list

Same flow for Dan.com (cheaper transfer fees, faster). Afternic is owned by GoDaddy and gets premium-domain reseller distribution.
