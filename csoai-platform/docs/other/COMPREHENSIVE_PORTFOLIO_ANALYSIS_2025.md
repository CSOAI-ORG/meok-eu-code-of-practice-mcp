## ANALYSIS FRAMEWORK

This analysis evaluates each company across multiple dimensions:

### 1. VALUATION METHODOLOGY

**Base Valuation Factors:**
- Domain value (£5K - £35K per domain)
- Development status (Alpha/Beta/Production)
- Technical readiness score (0-40 points)
- Marketing readiness score (0-30 points)
- Content completeness score (0-20 points)
- Market size and TAM (Total Addressable Market)
- Competitive positioning and USP strength

**Valuation Multipliers:**
- Revenue (ARR × 5-10x for SaaS, 15-25x for AI Safety)
- Strategic positioning (TC260 compliance, DWT 2026 mandate)
- Network effects (ecosystem integration)
- IP value (RLMAI backend, provisional patents)

### 2. PRODUCTION READINESS ASSESSMENT

**Scoring System (0-100):**
- **Technical (40 points):** Backend, frontend, payments, hosting, security
- **Marketing (30 points):** Brand, SEO, content, social proof
- **Content (20 points):** Copy, images, documentation
- **Legal/Ops (10 points):** Terms, privacy, support

**Readiness Stages:**
- **Alpha (0-40):** Concept/prototype, not market-ready
- **Beta (41-80):** Core features working, needs polish
- **Production (81-100):** Fully functional, revenue-ready

### 3. STRATEGIC PRIORITY MATRIX

**Priority Tiers:**
- **Tier 1 (Focus Now):** High value + High readiness + Market timing
- **Tier 2 (Build Next):** High value + Medium readiness
- **Tier 3 (Backlog):** Medium value or Low readiness
- **Tier 4 (Divest/Merge):** Low value + Low readiness
